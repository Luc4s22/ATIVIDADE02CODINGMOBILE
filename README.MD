# API do Drink 🍹

Este projeto é um aplicativo que consome a **API do Drink** para mostrar informações sobre drink e suas receitas. 
## Tecnologias

- HTML / CSS / JavaScript (Front-end)
- API:  (`https://www.thecocktaildb.com/api/json/v1/1/search.php)
- Fetch API para requisições HTTP

## Funcionalidades

- Buscar drinks por nome
- A foto do drink vai aparecer
- Mostrar ingredientes e instruções de preparo
- Interface simples e interativa para explorar bebidas

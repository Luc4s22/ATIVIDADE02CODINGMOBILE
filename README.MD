# API do Drink 🍹

## Objetivo do Projeto
O objetivo deste projeto é mostrar como consumir uma **API pública** usando **JavaScript**, fazer requisições HTTP e mostrar os dados de forma prática na tela. Além disso, serve pra treinar front-end e manipulação de dados de uma API real.

## Tecnologias
- HTML / CSS / JavaScript (Front-end)  
- API: https://www.thecocktaildb.com/api.php
- Fetch API para requisições HTTP  
- Netlify (deploy da aplicação)

## Funcionalidades
- Buscar drinks pelo nome  
- Exibir a foto do drink  
- Mostrar ingredientes e instruções de preparo  
- Interface simples e interativa  

## Como Acessar
A aplicação está online: (https://shiny-unicorn-b4257d.netlify.app/)

## Aprendizados
- Consumir APIs públicas e trabalhar com JSON  
- Criar interfaces simples e funcionais  
- Fazer deploy no Netlify
